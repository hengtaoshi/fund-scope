## 一、项目概述

个人基金量化分析平台，通过多因子评分模型对基金进行综合评价，提供买卖参考建议。

### 技术栈

| 层 | 技术 |
|------|------|
| 后端 | Python Flask |
| 数据源 | akshare（天天基金/东方财富公开数据） |
| 数据库 | SQLite（持仓存储） |
| 前端 | 纯 HTML/CSS/JS（无框架） |
| 图表 | Chart.js |
| 邮件 | smtplib（163 邮箱） |

### 启动方式

```bash
cd quantitative-warehouse
pip install flask akshare pandas numpy
python backend/app.py
# 浏览器打开 http://localhost:5000
```

## 五、多因子评分模型

```
综合评分 = 收益能力(30%) + 风控能力(25%) + 性价比(20%) 
         + 技术面(15%) + 稳定性(10%)
```

### 评分等级

| 分数 | 等级 | 操作建议 |
|------|------|---------|
| 90-100 | ⭐⭐⭐⭐⭐ 优质 | 强烈推荐买入 |
| 70-89 | ⭐⭐⭐⭐ 良好 | 推荐买入 |
| 50-69 | ⭐⭐⭐ 一般 | 观望 / 定投 |
| 30-49 | ⭐⭐ 较差 | 建议减仓 |
| 0-29 | ⭐ 差质 | 建议卖出 |

### 技术信号

| 指标 | 作用 |
|------|------|
| RSI（14） | 判断超买超卖 |
| MACD（12/26/9） | 判断动量方向 |
| 均线系统（5/20/60日） | 判断趋势 |
| 布林带（20,2σ） | 判断价格位置 |
