# 基金驾驶舱 · 开发方案

> 个人基金量化分析平台 — 数据驱动决策，告别凭感觉买基金

---

## 一、项目定位

**一句话**：帮自己看清持仓、分析基金、获得买卖参考的个人网站。

**三个原则：**
1. ✅ 只分析，不下单（决策权在自己手里）
2. ✅ 用数据说话，不搞玄学预测
3. ✅ 个人使用，不商用，不碰法律红线

---

## 二、技术方案

| 层 | 技术 | 理由 |
|---|---|---|
| 前端 | HTML/CSS/JS 纯静态 Demo → 后续转 Streamlit | 快速出原型，再转可运行应用 |
| 数据源 | **akshare**（免费开源，覆盖天天基金数据） | 合法、免费、一行代码拿数据 |
| 后端 | Python Flask（后续阶段） | 发邮件、数据处理 |
| 数据库 | SQLite（后续阶段） | 存持仓、缓存数据 |
| 图表 | Chart.js（现阶段）→ ECharts（后续） | 交互式图表 |

## 四、数据方案

```python
import akshare as ak

# 基金历史净值
df = ak.fund_open_fund_info_em(symbol="005827", indicator="单位净值走势")

# 基金基本信息
info = ak.fund_info_em(symbol="005827")

# 基金经理
manager = ak.fund_manager_em(symbol="005827")
```

### 法律合规

- ✅ 仅获取公开数据（浏览器可查看的内容）
- ✅ 个人使用，不商用，不分发
- ✅ 低频请求（请求间隔 ≥ 1 秒）
- ❌ 不突破反爬机制
- ❌ 不造成对方服务器过载
- ❌ 不转发/出售数据

## 五、多因子评分模型

### 公式

```
综合评分 = 收益能力(30%) + 风控能力(25%) + 性价比(20%) 
         + 基金经理(15%) + 基金公司(10%)
```

### 评分等级

| 分数 | 等级 | 操作建议 |
|------|------|---------|
| 90-100 | ⭐⭐⭐⭐⭐ 优质 | 强烈推荐买入 |
| 70-89 | ⭐⭐⭐⭐ 良好 | 推荐买入 |
| 50-69 | ⭐⭐⭐ 一般 | 观望 / 定投 |
| 30-49 | ⭐⭐ 较差 | 建议减仓 |
| 0-29 | ⭐ 差质 | 建议卖出 |

### 技术信号

| 指标 | 作用 |
|------|------|
| RSI（相对强弱指数） | 判断超买超卖（<30 超卖，>70 超买） |
| MACD | 判断动量方向（金叉↑买入，死叉↓卖出） |
| 均线系统（5/20/60日） | 判断趋势（多头排列↑持有，空头排列↓观望） |
| 布林带 | 判断价格位置（下轨支撑，上轨压力） |
